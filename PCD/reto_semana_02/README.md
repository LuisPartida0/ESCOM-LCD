# Clasificador de Temperaturas (Reto 02)

## Descripción
Este programa estandariza reportes de clima internacionales. Convierte temperaturas de Fahrenheit a Celsius, valida los datos de entrada y clasifica cada ciudad en categorías que van desde "Congelante" hasta "Extremo".

## Instalación y Uso
1. Asegúrate de tener Python 3.x instalado.
2. Ejecuta el programa redirigiendo un archivo de texto o mediante entrada manual:

```bash
python main.py < tests/entrada1.txt
